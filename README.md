# Muscle Ageing Analysis Dashboard

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

## Overview
This is a **Shiny for Python** application designed to visualise transcriptomic data and machine learning results related to human skeletal muscle ageing. It accompanies the research presented in **Chapter 2** of Karen Guerrero-Vázquez's PhD Thesis.

## Installation
To run this app locally:

1. Clone this repository.
2. Install dependencies:
   ```bash
   pip install shiny pandas plotly
   ```
   
### Run the app:

```
shiny run app.py
```
